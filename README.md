## Featured Projects

### Switch Yard
Switch Yard is an application for creating virtual smart home switches, enabling users to control smart devices through cloud-based infrastructure. The backend is powered by AWS Lambda, with authentication managed by AWS Cognito and data stored in AWS DynamoDB. The app currently integrates with Alexa for smart home control, with plans to support additional platforms in the future.
<br>**Technologies:** AWS Lambda, AWS Cognito, AWS DynamoDB, React, Alexa Integration
<br>**Status:** In Progress

### [TikTok Engagement Analysis](https://github.com/lewi6099/tiktok-engagement)
This project investigates the relationship between TikTok video length and total user interactions (likes, shares, comments, downloads). Using a public dataset, the analysis includes data cleaning, exploratory data analysis, and statistical testing to determine if video duration impacts engagement. The project is implemented in a Jupyter Notebook using Python and Pandas for data manipulation and analysis.
<br>**Technologies:** Python, Pandas, Jupyter Notebook
<br>**Completed:** Fall 2025
